# DogoGram

## Description
Whole my life.. I wanted to try ant.design using React.js =)

## Demo
`https://bs-dogogram.herokuapp.com`

## How to start?
* git clone https://github.com/black-stork/dogogram.git
* cd dogogram
* npm install
* npm run start
* open http://localhost:8080/
